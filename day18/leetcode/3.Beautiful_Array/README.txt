Массив nums длины n красив, если:

nums представляет собой перестановку целых чисел в диапазоне [1, n].
Для каждого 0 <= i < j < n нет индекса k с i < k < j где 2 * nums[k] == nums[i] + nums[j].
Учитывая целое число n, верните любой красивый массив nums длины n. На данный вопрос будет хотя бы один правильный ответ n.

 

Пример 1:

Ввод: n = 4 
Вывод: [2,1,4,3]
Пример 2:

Ввод: n = 5 
Вывод: [3,1,2,5,4]
